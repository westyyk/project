# Task Manager - приложение для учета задач с дедлайнами и категориями #

Веб-сайт с авторизацией для управления задачами, предназначенный для студентов и разработчиков. Позволяет создавать задачи, распределять их по категориям, указывать дедлайны и отслеживать статус выполнения.

## Функциональность

**Управление задачами**

- *Создание новой задачи*
- *Редактирование*
- *Удаление*
- *Отметка выполнения*
- *Поиск по названию*

**Категории**

- *Создание собсветнных категорий*
- *Назначение категории каждой задаче*
- *Фильтрация задач по категории*

**Дедлайны**

- *Указание даты завершения*
- *Подсветка просроченных задач*
- *Сортировка по близости дедлайна*

 **Интерфейс** 

- *Удобный список задач*
- *Панель категорий*
- *Статусы: В процессе, Готово, Просрочено*
- *Поддержка тёмной/светлой темы*

**Технологии**

Проект реализован с использованием:
- *Язык: Python*
- *База данных: SQlite*
- *Веб-фреймворк Flask*
